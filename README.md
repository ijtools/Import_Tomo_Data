# Import Tomo Data
Simple plugin to import 3D volume images as generated by X-ray microtomography synchrotron beamlines.

## Installation

Download the ".jar" file corresponding to the latest release, and copy the file into the "plugins" directory 
of your ImageJ / Fiji installation.
When restarting Fiji, you should have a new menu entry "File -> Import -> Tomo Data"

## Usage

Simply run the plugin, and choose the ".vol" file corresponding to the tomography data.
The plugin identified the corresponding vol.info file, retrieves the meta data required for reading,
import the volume, and displays it into a new frame.
